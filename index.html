Descripción:
Aplicación web desarrollada en HTML, CSS y JavaScript, diseñada para optimizar la administración de gimnasios mediante una interfaz amigable y funcionalidades clave para el control diario.

  • ✅ Gestión de membresías flexibles: diaria, semanal, quincenal, mensual, trimestral, cuatrimestral, semestral y anual.
  • ✅ Generación y envío de códigos QR personalizados para el acceso al gimnasio.
  • ✅ Registro automático de asistencias mediante escaneo QR.
  • ✅ Administración de cobros y reportes financieros de membresías.
  • ✅ Panel de control para registrar y gestionar usuarios del sistema.
  • ✅ Personalización visual: permite configurar el nombre del gimnasio, colores, logo y fondo.

Tecnologías utilizadas:
HTML5, CSS3, JavaScript (vanilla)

Este proyecto busca brindar una solución accesible y adaptable para gimnasios que desean digitalizar sus procesos de control de usuarios y accesos sin necesidad de plataformas externas.
